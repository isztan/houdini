Houdini

A simple collapse-and-expand widget.

How It Works

Hide content with the .collapse class.

Use any <a> or <button> element to show and hide collapsed content by adding the .collapse-toggle class. You also need to include a data-target attribute that matches the ID of the collapsed content.

Now you see me, now you don't.

Show + Hide - 

<div class="collapse" id="show-me">
    <p>Now you see me, now you don't.</p>
</div>

<a class="collapse-toggle" data-target="#show-me" href="#">
    <span class="collapse-text-show">Show +</span>
    <span class="collapse-text-hide">Hide -</span>
</a>

Show Content by Default

If you'd prefer to show content by default, include the .active class along with the .collapse and .collapse-toggle classes.

Hide me!

<button class="collapse-toggle active" data-target="#hide-me">
    <span class="collapse-text-show">Show</span>
    <span class="collapse-text-hide">Hide</span>
</button>

<div class="collapse active" id="hide-me">
    <p>Hide me!</p>
</div>

Progressively Enhanced

By default, .collapse-toggle elements are hidden and .collapse elements are visible.

Houdini includes a script that checks for JavaScript support by adding a .js class to the <body> on page load. This activates Houdini, showing the toggles and hiding collapsed content.

That way, people without JavaScript support still have access to the content.

Browser Support

Houdini works in all modern browsers and IE 6 and above. Houdini also requires jQuery.
